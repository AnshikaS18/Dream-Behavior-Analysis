# Dream Behavior & Emotional Analysis

This project analyzes dream records to understand how stress levels,
sleep stages, and emotional states influence dream sentiment and behavior.

## Dataset Source

The dataset used in this project was self-generated through web scraping.
Dream entries were collected from publicly available online sources such as
Reddit discussion threads and online dream journal websites.

The raw data was cleaned, structured, and transformed into an analysis-ready
format before being stored in a MySQL database.

No personally identifiable information was collected or retained.
